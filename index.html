CSS Cheat Sheet

What is CSS?

CSS is the code that styles a website. CSS stands for Cascading Style Sheets, and describes how the browser should represent the content of the HTML file. The CSS file ends with the extension .css.

CSS and HTML

CSS code is used to style HTML content in three ways:

  1. Within the HTML tag using the style attribute
  2. Within the HTML file (generally within the head tag) enclosed by the style tag.
  3. Within a seperate file with the .css extension.
Generally best practice is to seperate the CSS code and the HTML code, so that the website is organized in a modular fashion. This makes for easier maintenance and scalability. When an external CSS file is used, it must be linked to the HTML file using the link tag with the attributes: href="/path/to/file" rel="stylesheet" type="text/css".

CSS Code Structure

CSS code is structured by a selector and a declaration block that contains property/value pairs. An example is: selector { property-one: value; property-two: value; } The curly bracket characters enclose the declaration block. Each property value controls the way that the item denoted by the selctore is displayed in the browser. The selector can be an HTML tag, class, id, or a combination of these. For example .class-name h1 denotes all the h1 tags with the class attribute class-name. The more specific selector overrides less specific selectors (for example, an id selector overrides a tag selector); however, the hierarchy can be changes using the !important rule declaration.

Common CSS Styling Properties

Some of the major styling that CSS is used for is the following:

Color
Both foreground and background colors can be controlled, along with their saturation, hue and lightness properties
Font
Different fonts can be selected, along with their properties such as size, weight, line positioning, word spacing and character spacing.
Positioning
The positioning of the content can be controlled through display, and spacing properties

Color

Color properties can define both foreground and background color of HTML content. This is done with the color and background-color properties of a object, respectively. Colors can be defined several ways: by a color name, a hexadecimal value, or an RGB value. In addition, transparency of a color can be set via the alpha value of the color degfinition.

PropertyDescription
colorColor of the foreground element
background-colorColor of the background element

Font

Fonts have several defined properties: weight, style, and family. Weight determines the how thick or thin characters are in a text. Style determines italic or normal (oblique is also an option, but not as supported). Finally, font family determines the actual typography that is used; there should always be a "web safe" fall-back option for these. However, fonts can be imported from websites such as Google Fonts, or stored locally by using the @font-face rule.

PropertyDescription
font-weightThinness and thickness of the character lines. For example: bold.
font-styleDetermines the italic or normal style.
font-familyDetermines the actual font style; for example, Helvetica or Times New Roman.

Positioning

Positing determines where the HTML elements are displayed on the screen. The default positing is static; however, there are also fixed, relative and absolute positining schemes. Fixed items remain in the same place regardless of user scrolling; relative items are displayed at a position relative to their original position; and, finally, absolute positions an element relative to its parent object's position. In addition to these schemes, there is the display property of an object which determines the render block of an element; these are block, inline and inline-block values. Block items take up the entire width of their container, whereas in line items take up as little horizontal space as possible. Inline-block items are able to be placed next to each other in a horizontal row.
